# 九食堂面包店在线订购系统

一个功能完整的面包店在线订购系统，支持用户端和商家端双模式操作。

## 🌟 功能特色

### 用户端功能
- 🛍️ **商品浏览** - 查看新鲜面包，支持口味选择
- 🛒 **购物车管理** - 添加、删除、修改商品数量
- 📋 **订单管理** - 查看历史订单和订单状态
- 👤 **个人中心** - 管理收货地址和订单备注
- 💾 **本地存储** - 数据自动保存，刷新不丢失

### 商家端功能
- 📊 **数据统计** - 今日销售额、订单数、库存统计
- 📦 **库存管理** - 添加、编辑、删除商品，实时库存管理
- 📋 **订单处理** - 接单、配送、完成订单流程管理
- ⚙️ **系统设置** - 店铺信息配置

## 🚀 快速开始

### 方法一：直接打开
直接双击 `index.html` 文件在浏览器中打开

### 方法二：本地服务器
```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
npx serve .

# 使用PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

## 📱 使用说明

### 用户操作流程
1. 打开网站，默认进入用户端
2. 浏览商品列表，选择心仪的面包
3. 选择口味（如有多种口味）
4. 点击"加入购物车"
5. 进入购物车页面确认订单
6. 点击"去结算"进入个人中心
7. 填写收货地址和订单备注
8. 提交订单等待商家处理

### 商家操作流程
1. 点击右上角切换到商家端
2. 查看数据统计了解经营状况
3. 在库存管理中添加或编辑商品
4. 在订单管理中处理用户订单
5. 及时更新订单状态

## 🎨 自定义配置

### 修改商品信息
在 `index.html` 的 JavaScript 部分找到 `products` 数组，可以修改商品信息：

```javascript
products: [
    {
        id: 1,
        name: '商品名称',
        description: '商品描述',
        price: 价格,
        emoji: '🍞',
        stock: 库存数量,
        image: '图片URL',
        // ...
    }
]
```

### 修改主题色彩
在 CSS 部分的 `:root` 选择器中修改色彩变量：

```css
:root {
    --primary-color: #FFD93D;    /* 主色调 */
    --secondary-color: #FFB347;   /* 辅助色 */
    --accent-color: #FF6B6B;     /* 强调色 */
    // ...
}
```
